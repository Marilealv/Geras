O **Geras** é uma plataforma desenvolvida para facilitar e mediar o processo de doações para asilos da região. Atuamos como uma ponte entre o doador e o donatário, ajudando a identificar as necessidades reais e as particularidades de cada instituição para que cada doação tenha o máximo impacto positivo.


## Sobre o Projeto

O objetivo principal do Geras é humanizar e organizar o fluxo de ajuda, garantindo que as instituições recebam exatamente o que precisam. Através da plataforma, buscamos:
* **Auxiliar** no caminho de doações para asilos locais.
* **Atuar como mediador** na relação entre quem doa e quem recebe.
* **Reconhecer e dar visibilidade** às necessidades específicas de cada idoso e instituição.


## Tecnologias

Este projeto foi construído utilizando as seguintes tecnologias:
* **Node.js** & **npm**
* **Vite** (ferramental de desenvolvimento e build rápido)


## Como Instalar e Executar

Para rodar o projeto localmente em sua máquina, siga os passos abaixo:

### Pré-requisitos
É necessário ter o **Node.js** e o **npm** (Node Package Manager) instalados em seu sistema.

### Passo a Passo

1. **Instalar as dependências:**
   Abra o terminal na pasta raiz do projeto e execute:
   ```bash
   npm install
   ```
2. **Gerar o build do projeto:**
   Para compilar e otimizar os arquivos para produção, utilize:
   ```bash
   npm run build
   ```
3. **Executar localmente:**
   Para iniciar o servidor de desenvolvimento e visualizar o site, use o comando:
   ```bash
   npx vite
   ```
